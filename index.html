avaje jvm libraries

Config

Provides external configuration for applications. Loads yaml and properties files, supports dynamic configuration

Metrics

APM style metrics for JVM with fast and light timing metrics with @Timed, gauges, counters with built in JVM and CGroup metrics.

DInject

Dependency injection via code generation for server side developers. Similar feature set to Spring DI but uses APT code generation (like Dagger2).

Javalin
controllers

Code generation for Javalin controllers such that we can have JAX-RS style controllers with @Path, @Get etc but extremely light and fast.

Kotlin KAPT

Maven tiles to make it easier to use Kotlin KAPT with maven.

Maven tiles

Useful maven tiles to drive reuse in maven build definitions.
